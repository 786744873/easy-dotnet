# 最喜欢的单行代码

你最喜欢的 JavaScript 单 LOC（代码行）是什么？让我知道！

## 在本地运行它

1. 克隆项目：

```shell
$ git clone git@github.com:phuocng/1loc.git
```

1. 安装依赖项：

```shell
$ cd 1loc
$ npm install
```

1. 在本地运行它：

```shell
$ npm run start
```

在浏览器中打开[http://localhost:8081以查看它的运行情况。](http://localhost:8081/)
