## 要求

Node >=12.x <15

## 用法

1. 安装必要的节点模块。

```
npm i
```

1. 运行 npm 脚本 `start`开始开发。

```
npm run start
```

1. 完成开发后，运行 npm 脚本 `build`制作压缩文件。

```
npm run build
```
