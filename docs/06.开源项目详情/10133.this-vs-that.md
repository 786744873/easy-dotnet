# this-vs-that

前端开发中 ***和\*** 的区别。

- [X] 涵盖各种主题（CSS、DOM、HTML、JavaScript）的差异
- [X] 包括许多*有用的知识*部分
- [X] 包括许多好的做法
- [X] 包括许多提示和技巧

## 在本地运行它

1. 克隆项目：

```shell
$ git clone https://github.com/phuocng/this-vs-that
```

1. 安装依赖项：

```shell
$ cd this-vs-that
$ npm install
```

1. 在本地运行它：

```shell
$ npm run start
```

在浏览器中打开[http://localhost:8081以查看它的运行情况。](http://localhost:8081/)
