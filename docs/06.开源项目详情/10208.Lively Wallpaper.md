## 关于
<img src="/img/open/10208/main_preview.gif" width="600" height="338"/>

Lively 是您唯一需要的动画桌面壁纸应用程序。

<img src="/img/open/10208/main_dragdrop.gif" width="600" height="338"/>

它超级简单.. 几乎没有不便，只需拖放文件或网页即可设置为墙纸。

## 特征
*稍等一下，预览 gif 剪辑需要一些时间才能加载。*
#### 视频 / GIFs
<img src="/img/open/10208/wallpaper_video.gif" width="600" height="338"/>

<a href="https://visualdon.uk/project/eternal-light/">VISUALDON的永恒之光</a>
* 由 mpv / vlc 播放器提供支持。
* 使用过滤器自定义视频。
* 支持硬件加速。
* 支持整数缩放器以获得清晰的图像。
#### 网页
<img src="/img/open/10208/wallpaper_html.gif" width="600" height="338"/>

<a href="http://louie.co.nz/25th_hour/"> 路易斯·科伊尔（Loius Coyle）的第 25 小时</a>
* 加载 HTML 文件或网址作为壁纸。
* 运行 webgl、javascript .. 基本上任何适用于 chrome 的东西。
* 音频反应壁纸支持。
#### 着色器
<img src="/img/open/10208/wallpaper_shadertoy.gif" width="600" height="338"/>

<a href="https://www.shadertoy.com/view/ltffzl">BigWings的衷心 </a>
* 在浏览器中运行 GLSL 着色器。
* 自定义着色器渲染器（即将推出。）
* 支持 Shadertoy.com 网址作为墙纸。
#### 游戏和应用程序
<img src="/img/open/10208/wallpaper_emulator.gif" width="600" height="338"/>

* 支持 Unity、Godot 游戏作为壁纸。
* 对 gui 应用程序作为壁纸的有限支持。
* [学到更多..](https://github.com/rocksdanister/lively/wiki/Application-Wallpaper)
#### 屏保
<img src="/img/open/10208/screen_saver.gif" width="600" height="338"/>

* Lively 支持 Windows 屏幕保护程序。
* 任何壁纸都可以用作屏幕保护程序。
* 支持多显示器。
* [学到更多..](https://github.com/rocksdanister/lively/wiki/Screen-Saver)
#### 自动化
<img src="/img/open/10208/cmd_rm.gif" width="600" height="338"/>

* Lively 可以通过终端的命令行参数进行控制。
* 可以与 AutoHotkey 等第三方脚本软件配对。
* [学到更多..](https://github.com/rocksdanister/lively/wiki/Command-Line-Controls)
#### 活泼的 API
<img src="/img/open/10208/music_api.gif" width="600" height="338"/>

* 强大的 API 集供开发人员创建交互式壁纸。
* 获取系统信息、音频图、音乐信息..
* [学到更多..](https://github.com/rocksdanister/lively/wiki/API)
#### & 更多的
便于使用; 只需将媒体文件和网页拖放到生动的窗口中即可将其设置为墙纸。
硬件加速视频播放，由 mpv 播放器提供支持。
交互式 webgl 壁纸，由轻量级铬引擎 Cef 提供支持。
Windows 11 流利的设计语言。
高效，Lively 是一个原生的 .net 核心应用程序。
完全开源且免费；没有黑魔法，没有付费墙后面的功能。
#### 表现
当机器上运行全屏应用程序/游戏时，壁纸播放暂停（~0% cpu，gpu 使用率）。
根据正在运行的前台应用程序设置壁纸播放规则。
可以设置为使用电池供电（笔记本电脑）运行时暂停壁纸。
运行远程桌面会话时，墙纸会自动暂停。
#### 多显示器
支持任意数量的显示器。
壁纸会根据显示器设置的变化自动调整。
在所有屏幕上跨越单个壁纸。
在所有屏幕上复制相同的壁纸。
支持 HiDPI 和超宽分辨率。

**_我没有正式隶属于 Unity 技术、godot、shadertoy；_**
