## 快速开始

### 使用 docker-compose 运行

```shell
mkdir answer && cd answer
wget https://raw.githubusercontent.com/answerdev/answer/main/docker-compose.yaml
docker-compose up
```

有关更多信息，您可以查看[INSTALL.md](https://github.com/answerdev/answer/blob/main/INSTALL.md)
