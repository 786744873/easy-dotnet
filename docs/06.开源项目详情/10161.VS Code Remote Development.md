# Visual Studio Code 远程开发

此存储库用于提供有关**[Visual Studio 远程开发](https://aka.ms/vscode-remote/download/extension)**扩展包及其相关扩展的反馈。您可以使用存储库报告问题或提交任何这些扩展的功能请求：

- [远程 - SSH](https://aka.ms/vscode-remote/download/ssh)
  - 远程 - SSH：编辑配置文件
- **[远程 - 容器](https://aka.ms/vscode-remote/download/containers)**
- **[远程 - WSL](https://aka.ms/vscode-remote/download/wsl)**

您还可以使用此存储库来提供有关**[Visual Studio Code Server](https://aka.ms/vscode-server-doc)**的反馈。

如果您想与远程开发扩展一起使用的 **另一个扩展遇到问题，请在扩展的存储库中提出问题。** 您可以参考我们的远程相关问题提示[摘要](https://aka.ms/vscode-remote/troubleshooting/extensions)和我们的[扩展指南](https://aka.ms/vscode-remote/developing-extensions)，以帮助扩展作者入门。

可以在[vscode-dev-containers 存储库](https://aka.ms/vscode-dev-containers)中报告与开发容器定义相关的问题。您可以使用[开发容器规范存储库](https://github.com/microsoft/dev-container-spec)来归档和审查问题，以制定开发容器和开发容器 CLI 的方向。

## 发布

VS Code 远程开发扩展“稳定”版本直接绑定到 VS Code 版本。发布亮点可以在[VS Code 发布说明](https://code.visualstudio.com/updates)中找到，其中将包含指向[详细扩展发布说明](https://github.com/microsoft/vscode-docs/tree/main/remote-release-notes)的链接。

扩展使用与VS Code 本身相同的[开发过程和时间表进行开发。](https://github.com/microsoft/vscode/wiki/Development-Process#inside-an-iteration)[您可以在此存储库中的固定规划问题中](https://github.com/microsoft/vscode-remote-release/issues)查看当前开发迭代的计划。本期将包含一个指向更广泛的 VS Code 计划的链接。与 VS Code 本身一样，扩展将在开发迭代期间更新，其中的更改仅在[VS Code Insiders Edition](https://code.visualstudio.com/insiders/)中可用。
