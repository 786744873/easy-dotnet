# 让我们开始吧

首先，您需要 `fl_chart`在项目中添加。为此，请遵循[本指南](https://pub.dev/packages/fl_chart/install)。

然后你需要阅读文档。从[这里](https://github.com/imaNNeoFighT/fl_chart/blob/master/repo_files/documentations/index.md)开始。

我们建议您查看示例源代码。
